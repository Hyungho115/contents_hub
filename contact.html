GitHub Issues 열기 개인정보 안내 보기
  • 현재 채널 : GitHub Issues
  • 권장 용도 : 오류 제보, 수정 요청, 운영 제안
  • 주의 사항 : 민감한 개인정보는 공개 채널에 남기지 않기

문의 가능 범위

이런 문의를 받을 수 있습니다

  • 콘텐츠 수정 요청: 영화 정보, 감상 순서, 링크 설명 등에서 사실 확인이 필요한 경우
  • 오류 제보: 깨진 링크, 잘못된 이동 경로, 페이지 표시 문제를 발견한 경우
  • 운영 제안: 다뤄줬으면 하는 프랜차이즈, 허브 구조 제안, 탐색 방식 개선 의견
  • 정책 관련 문의: 광고, 개인정보처리방침, 공개 운영 기준과 관련된 질문이 있는 경우

문의 품질 가이드

문의할 때 함께 적어 주세요

  • 대상 페이지 URL: 어떤 페이지에 대한 문의인지 바로 확인할 수 있게 적어 주세요.
  • 문제 내용: 무엇이 잘못되었는지 또는 어떤 점을 개선하고 싶은지 구체적으로 적어 주세요.
  • 기대하는 수정 방향: 사실 정정, 문장 보완, 링크 수정처럼 원하는 조치가 있으면 함께 적어 주세요.

공개 채널 안내

개인정보와 비공개 문의 안내

현재는 공개 채널만 운영하고 있으므로, 민감한 개인정보나 비공개 자료를 포함한 문의는 보내지 말아 주세요. 별도의 비공개 연락 수단이 준비되면 이 페이지에 즉시 반영하겠습니다.

운영 관련 문의, 수정 요청, 링크 오류 제보는 GitHub Issues 에서 남길 수 있습니다.
